口型同步
===================

口型同步图片必须存放在“lip”文件夹中，该文件夹需与角色的立绘图片位于同一目录下。

* happy.png （角色主文件）
* lip/
    * happy.png （口型同步文件）

口型同步图片由一帧或多帧表示不同口型的图像组成。

每一帧的尺寸必须与角色立绘的尺寸完全相同。
各帧必须按照从左到右的顺序水平排列。

边缘的 Alpha 值（不透明度）必须进行平滑处理。
请在图像编辑软件中使用“羽化选区”和“删除选区”功能。

口型同步的间隔时间可以在 `project.txt` 文件中进行指定。

```
#
# 口型同步帧长（每帧持续的秒数）
#
character.lipsync.frame=0.3
#
# 每个字符对应的口型同步次数
#
character.lipsync.chars=3
```
