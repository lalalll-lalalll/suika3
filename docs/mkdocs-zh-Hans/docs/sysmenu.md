系统菜单
===========

## 系统按钮

Suika3 配备了一个汉堡菜单按钮，通常位于屏幕的左上角。这个按钮在内部被称为`系统按钮`。

为了确保符合应用商店的准则，Suika3 避免在对话框周围放置类似 PC 端的小按钮，而是采用了“移动端优先”的设计思路。系统按钮由两张图片和相关的动画组成。为了提供流畅的用户体验，该按钮具有响应性：当鼠标移动或触摸屏幕时它会显现，并在几秒无操作后自动隐藏。虽然这种行为是为了符合商店规范而硬编码的，但你可以通过在配置文件中设置 `sysbtn.enable=false` 来完全禁用系统按钮——例如，用于演示版或 Kiosk 模式。

虽然在熟悉的对话框周围没有按钮起初可能会让人觉得不习惯，但我们希望你能理解，这种演变对于让视觉小说适应现代移动平台是必不可少的。

另请参阅：
- `config.ini` (搜索 `sysbtn`)
- `system/sysbtn/` (在示例游戏中)

## 系统菜单

点击系统按钮会触发一个被称为`系统菜单（SysMenu）`的图形用户界面。系统菜单是完全可定制的，你可以配置它来包含存档、读档、自动模式、跳过模式、历史记录和设置等核心功能。

另请参阅：
- `system/sysmenu/` (在示例游戏中)

## 存档/读档界面

存档和读档界面可以通过 GUI 文件进行完全定制。

另请参阅：
- `system/save/` (在示例游戏中)
- `system/load/` (在示例游戏中)

## 自动模式和跳过模式按钮

自动和跳过按钮使用了专门的 GUI 按钮类型。这些按钮在被点击时会触发各自的模式——自动模式或跳过模式。

另请参阅：
- `system/sysmenu/` (在示例游戏中)

## 历史记录界面

历史记录界面可以通过 GUI 文件进行完全定制。

另请参阅：
- `system/history/` (在示例游戏中)

## 设置界面

设置界面可以通过 GUI 文件进行完全定制。

示例游戏中的设置界面包含：
- 背景音乐、音效和语音音量滑块
- 语言切换（英文/日文）
- 文本速度和自动模式速度调整
- 文本速度预览

此外，你还可以实现：
- 总音量
- 每个角色的音量
- 语言切换（针对所有支持的语言）
- 自定义按钮

另请参阅：
- `system/config/` (in the sample game)
