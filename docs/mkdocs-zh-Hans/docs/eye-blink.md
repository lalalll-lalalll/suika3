眨眼效果
============

眨眼效果图片必须存放在“eye”文件夹中，该文件夹需与角色的立绘图片位于同一目录下。

- happy.png （角色主文件）
- eye/
    - happy.png （眨眼效果文件）

眨眼图片由一帧或多帧表示不同眼部状态的图像组成。
每一帧的尺寸必须与角色立绘的尺寸完全相同。
各帧必须按照从左到右的顺序水平排列。关于实际图片的样式，请参考示例游戏。

边缘的 Alpha 值（不透明度）必须进行平滑处理。
请在图像编辑软件中使用“羽化选区”和“删除选区”功能。

眨眼的间隔时间可以在 `config.ini` 文件中进行指定。
间隔时间会有轻微的随机化，有时会出现连续眨眼两次的情况。

```
#
# 眨眼间隔（秒）
#
character.eyeblink.interval=4.0
#
# 眨眼帧长（每帧持续的秒数）
#
character.eyeblink.frame=0.15
```
